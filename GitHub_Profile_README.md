## Featured projects

### Account Sharing Detection Model — Alfa-Business

Repository: [account-sharing-detection](https://github.com/Sav1nDm1tr11/account-sharing-detection)

Behavioral classification model for corporate internet banking sessions:

- detects **normal owner access**, **trusted account sharing**, and **possible compromise**;
- uses **42 behavioral features** grouped into 7 signal layers;
- final ensemble reaches **93% accuracy**, **0.89 F1-macro**, and **0.94 ROC-AUC**.

### Smoking Probability Prediction

Repository: [smoking-probability-prediction](https://github.com/Sav1nDm1tr11/smoking-probability-prediction)

Binary classification project based on medical and anthropometric data:

- **15,000 train** observations and **10,000 test** observations;
- feature engineering increased the feature set to **43 variables**;
- used tree-based models, Optuna, Cleanlab, expert models, and ensembling;
- best honest OOF ROC-AUC: **0.8889**, public leaderboard score: **0.8880**.

### AI Citation Prediction for P&G Products

Repository: [ai-citation-prediction](https://github.com/Sav1nDm1tr11/ai-citation-prediction)

Predicting whether product pages are likely to be cited in generative AI search answers:

- worked with **218 observations** and **62 content / structural / SEO features**;
- built an interpretable classification pipeline with RFECV, logistic regression, and Optuna;
- achieved **AUC = 0.74** and identified content factors related to AI visibility.

### Inflation, GDP & Happiness

Repository: [inflation-gdp-happiness](https://github.com/Sav1nDm1tr11/inflation-gdp-happiness)

Statistical analysis of economic indicators and subjective well-being:

- combined World Happiness and macroeconomic data;
- analyzed **137 countries** in a 2019 cross-section;
- used descriptive statistics, distribution fitting, hypothesis testing, outlier detection, and correlation analysis;
- found GDP per capita to be the strongest predictor of happiness in the analyzed sample.

### X5 / Perekrestok Dashboard

Repository: [Cup-Russia-2026](https://github.com/Sav1nDm1tr11/Cup-Russia-2026)

Interactive business dashboard built for a case championship:

- React + Recharts dashboard for X5 / Perekrestok;
- visualizes P&L, NPV, CAPEX, EBITDA margin, ARPU, CAC, LTV, and revenue scenarios;
- connects financial modeling with BI-style storytelling.
